# Student Result Management System (SRM)

A Spring Boot web application used to manage students, subjects, and their exam results.  
The system allows administrators to add students, assign marks, generate results, and view consolidated performance reports.

---

##  Features

###  Student Management
- Add, update, delete student details
- View list of students

###  Subject Management
- Add, update, delete subjects
- Assign unique subject codes

###  Result Management
- Enter marks for students
- Automatic grade calculation
- View result per student or subject

###  Reports
- Generate result sheets
- View overall performance

---

##  Tech Stack

- **Spring Boot**
- **Spring Web / Spring Data JPA**
- **MySQL**
- **Maven**
- **Java 17+**
- **Swagger**
